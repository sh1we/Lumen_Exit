## О проекте (Overview)

**Lumen_Exit()** — это процедурный Survival Horror движок с псевдо-3D графикой, написанный с нуля на C++.

Проект является реализацией технологии **Raycasting** (метод бросания лучей), популяризированной в *Wolfenstein 3D*. В отличие от использования готовых движков (Unity/Unreal), здесь весь конвейер рендеринга, физика коллизий и генерация геометрии реализованы вручную через математические алгоритмы и прямую работу с пиксельным буфером.

**Цель:** Найти выход из бесконечно генерируемого темного лабиринта, ориентируясь только на источники света.

## Техническая реализация (Under the Hood)

Проект демонстрирует навыки системного программирования и оптимизации.

### 1. Кастомный Raycasting движок
*   **Алгоритм DDA (Digital Differential Analyzer):** Высокопроизводительный расчет пересечения луча с сеткой карты без использования "тяжелых" тригонометрических вычислений на каждом шаге.
*   **Коррекция Fish-Eye:** Устранение искажений перспективы через проекцию луча на плоскость камеры.
*   **Буфер отрисовки:** Прямая манипуляция массивом пикселей для отрисовки вертикальных полос стен.

### 2. Система освещения и шейдинг
*   **Distance-based Shading:** Реализована модель затемнения по закону обратных квадратов. Чем дальше стена, тем ниже значение RGB пикселя, уходящее в полный `#000000`.
*   **Атмосферный шум:** Симуляция зернистости и "тумана" для скрытия артефактов дальности прорисовки.

### 3. Процедурная генерация (RNG)
*   **Алгоритм Recursive Backtracker:** Генерация "идеальных" лабиринтов (без недостижимых зон) при каждом запуске.
*   **Seed-система:** Возможность воспроизвести конкретный уровень по сиду (опционально).

## Стек технологий (Tech Stack)

*   **Core:** C++17 (Smart Pointers, STL, Lambdas).
*   **IDE:** Microsoft Visual Studio 2022.
*   **Windowing/Input:** [SFML 2.6](https://www.sfml-dev.org/) — используется *исключительно* для создания окна и получения сырого буфера кадра. Вся 3D-математика — самописная.
*   **Math:** Тригонометрия и векторная алгебра.

| **Esc** | Выход в меню / Закрыть |
